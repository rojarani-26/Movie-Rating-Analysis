# Movie Ratings Analysis Using NumPy, Pandas, and Matplotlib

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a movie dataset to understand trends in movie ratings and analyze relationships between features such as **genre, director, budget, revenue, and year of release**.  
The analysis is performed using **Python libraries: NumPy, Pandas, and Matplotlib**.

## Objective
- Load and inspect a movie dataset using Pandas  
- Clean and preprocess the dataset  
- Perform statistical analysis using NumPy  
- Analyze movie ratings by **genre, director, and year**  
- Visualize trends using Matplotlib  
- Explore relationships between **budget, revenue, and ratings**

## Dataset Description
The dataset contains the following columns:

| Column | Description |
|------|-------------|
| Title | Name of the movie |
| Genre | Category of the movie (Action, Comedy, etc.) |
| Director | Director of the movie |
| Year | Year of movie release |
| Rating | Average audience rating |
| Budget | Budget used to produce the movie |
| Revenue | Total revenue generated by the movie |

## Tools and Technologies
- **Python**
- **NumPy** – Numerical computations
- **Pandas** – Data manipulation and analysis
- **Matplotlib** – Data visualization
- **Google Colab** – Development environment

## Project Workflow

### 1. Data Loading
The dataset is loaded using Pandas and inspected using:
- `head()`
- `info()`
- `isnull().sum()`

### 2. Data Cleaning
- Removed missing rating values
- Filled missing budget and revenue values with 0

### 3. Statistical Analysis
Using NumPy to compute:
- Mean
- Median
- Standard Deviation

for:
- Movie ratings
- Budget
- Revenue

### 4. Grouped Analysis
Using Pandas `groupby()` to analyze:
- Average rating by **Genre**
- Average rating by **Director**
- Average rating by **Year**

### 5. Data Visualization
Several visualizations were created using Matplotlib:

- **Bar Chart:** Average movie rating by genre
- **Bar Chart:** Average rating by director
- **Line Chart:** Movie ratings trend over years
- **Scatter Plot:** Budget vs Rating
- **Scatter Plot:** Revenue vs Rating
- **Horizontal Bar Chart:** Top 10 highest rated movies

## Key Insights
- The average movie rating in the dataset is around **7.33**.
- **Action, Mystery, and Sci-Fi** genres tend to have higher ratings.
- Some directors consistently produce **higher-rated movies**.
- Ratings vary slightly across different years.
- Higher budgets do not always guarantee higher ratings.
- Movies with higher ratings often generate **higher revenue**.

## Conclusion
This project demonstrates how Python libraries like **NumPy, Pandas, and Matplotlib** can be used for **data analysis and visualization**. By analyzing movie data, we can identify trends in audience preferences, genre popularity, and financial performance of movies.
